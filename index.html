Variant A: Combined Counter - Uses Event (Signal)
This variant combines both display and controls into a single component. Communication happens internally through events.
Variant B: Separated Counter - Uses Target
This variant separates display and controls. The controls component directly targets the display component by ID.
Standalone Components - Uses Event (Signal)
Fully separated display and controls components. Communication happens through custom events managed by external JavaScript.
Modified Component - Enhanced Features
Combined counter with added features: Undo (go back to previous value), Reset button, and adjustable step size (1, 5, or 10).
